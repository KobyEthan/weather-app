## Run Locally

Clone the project

```bash
  git clone https://github.com/KobyEthan/weather-app.git weather-app
```

Go to the project directory

```bash
  cd weather-app
```

Install dependencies

```bash
  npm install
```
Run project
```bash
  npm start
```
